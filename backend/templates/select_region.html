1단계: 시·도 선택
운동 추천에 사용할 지역의 대기질을 먼저 확인합니다.
Step 1 / 3
서비스 사용 흐름
  1. 시·도를 선택해 최신 대기질 정보를 확인합니다.
  2. 해당 시·도의 세부 측정소를 선택합니다.
  3. 사용자의 연령/BMI/성별/체력등급을 입력합니다.
  4. 대기질 수준을 반영한 실내/실외 추천과 운동 루틴을 확인합니다.
TIP
동일한 시·도라도 측정소별로 수치가 다르기 때문에,
실제 생활권과 가장 가까운 측정소를 선택하는 것이 좋습니다.
